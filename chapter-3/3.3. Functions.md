# 3.3. Functions

- シンプルな式

  `fn`を用いて関数を宣言することができる。

  ```rs
  fn main() {
    another_function();
  }
  fn another_function() {
    println!("Hello Another Function");
  }
  ```

- 引数

  ```rs
  fn main() {
    another_function(1);
  }
  fn another_function(x: i32) {
    println!("The Value X is {}", x);
  }
  ```

- 波括弧を用いた無名関数

  `{}`を利用してブロックを作ることで即時で実行される関数を作ることができます。文末にセミコロンがつけないことで値をリターンする。

  ```rs
  fn main() {
    let x = {
        let y = 3;
        y + 1
    }
  }
  ```
